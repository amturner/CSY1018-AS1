HTML

HyperText Markup Language (HTML) is a markup language that is used to create the structure of many of the websites you see on the web today.

For the basic structure of a website, HTML makes use of elements such as <html>, <head> and <body>. When describing content on a page, however, HTML also makes use of semantic elements such as <header>, <nav>, <main> and <footer>

As an example of how HTML code is typically structured, the following code shows you how to make a basic web page with the text "Hello World!":

<!DOCTYPE html>
<html>
    <head>
    </head>
    <body>
        <p>Hello World!</p>
    </body>
</html>

With these elements, HTML can therefore essentially be thought of as the skeleton of a web page.

CSS

Cascading Style Sheets (CSS) is a stylesheet language. Whilst HTML can be thought of as the skeleton of a web page, CSS can essentially be thought of as the skin of a web page.

Unlike HTML code, which makes use of elements like <header> and <footer>, CSS code is instead made up of selectors, which in turn contain properties.

As an example of how a typical CSS declartion is structured, the follow snippet of code shows you how you would make the background of a paragraph black, and the text white:

p {
    background-color: black;
    color: white;
}

JavaScript

JavaScript is a programming language that is commonly used by Web Developers to provide additional functionality on a web page.

Unlike HTML and CSS, which are a markup language and a stylesheet language respectively, the fact that JavaScript is a programming language means that JavaScript code can be executed outside of the user's web browser. As a result, this can give developers the ability to create standalone JavaScript applications.

Stylesheet

A Stylesheet is a document that is made up of CSS code. Stylesheets usually come in the form of files that end with the .css extension.

To add a Stylesheet to an HTML document, you should add this line between the <head> tags, as shown below:

<link href="style.css" type="text/css" rel="stylesheet">

Whilst the example shown here shows the Stylesheet "style.css" being imported from the website's root directory, a Stylesheet can be imported from a separate directory.

Directory

A Directory is the name for a store of files in within a File System. In an Operating System such as Windows, directories are more commonly known as Folders.

As a result of this, the terms Directory and Folder are often used interchangeably, regardless of the exact terminology that is used.

IDE

An Integrated Development Environment (IDE) is a piece of software that combines a text editor with tools specific to programming, such as linters, debuggers and compilers.

Alongside these tools, most IDEs also often support multiple programming languages with ease. In some cases, however, an IDE may only be designed with one programming language in mind.

Whilst code for any language can be written in pretty much any text editor (even Notepad!), text editors do not provide developers with the ability to properly format their code. In addition, a text editor such as Notepad does not include tools to assist in the debugging and/or linting of code, nor does it provide the ability to compile code into an executable program.

Some examples of IDEs include Visual Studio Code, Eclipse, JetBrains IntellJ IDEA and Code::Blocks.

Git

Git is an open-source version control system that is used for maintaining and keeping track of changes made to an application's source code.

Because Git is open-source, being licenced under version 2 of the GNU General Public License (GNU GPL), Git is free to use by anyone wishing to streamline how they manage a software development project.

When using Git, any files relating to a given project reside in a directory known as a repsitory. While repositories can reside on a local machine or a business's internal server, services such as GitHub and GitLab do exist to provide individuals or businesses with space to store their projects through the use of "cloud storage".
